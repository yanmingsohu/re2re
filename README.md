# BioHazard PC 1997 Reverse engineeringObjective: To compile the assembly code so that it can run.Current status: * Able to enter the first scene* The moving animation is strange* The program crashes randomly before reaching the title screen after opening.# Depends * vs2022 or h* ddisasm - https://github.com/GrammaTech/ddisasm# CompilingOpen the `cmd` command line to make the above tool `link/ddisasm` executable.Run `set > env.txt` in the project directory.`make` compiles and generates `bio2game/bio2.fixed.exe`.Other targets in `make` are used to disassemble and generate `bio2.fixed.S`; this is the initial workflow.
